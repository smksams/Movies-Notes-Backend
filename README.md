
# Movies notes
 Application in NodeJs made to create movie notes.



## Functionalities

- User registration and updating.
- Creating and editing notes by user
- Individual search filtered by tags and title


## Deploy

```bash
  npm install
```
```bash
  npm start
```
```bash
  npm run migrate
```
